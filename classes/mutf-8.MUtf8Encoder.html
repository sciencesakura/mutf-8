Class MUtf8Encoder

The encoder for Modified UTF-8.

Example

const encoder = new MUtf8Encoder();
const code = encoder.encode("Hello 世界!");
// Uint8Array [
//   0x48, 0x65, 0x6c, 0x6c, 0x6f, 0x20, 0xe4, 0xb8,
//   0x96, 0xe7, 0x95, 0x8c, 0x21,
// ]
